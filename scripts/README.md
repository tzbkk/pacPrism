# Scripts Directory

This directory contains CMake modules and configuration scripts used by the pacPrism build system.

## Files

- **AsioConfig.cmake** - asio dependency management and auto-fetching
- **PlatformConfig.cmake** - Platform-specific build configurations
- **BuildConfig.cmake** - General build configuration settings
- **LibraryConfig.cmake** - Library configuration functions and dependency management

## Usage

These modules are automatically included by the main CMakeLists.txt files in the project.
